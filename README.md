# Contentstack Cloud Engine — Nuxt demo

Single-page Nuxt 3 app matching the Launch deployment success screen.

## Setup

```bash
npm install
```

## Dev

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm run preview
```

Static export:

```bash
npm run generate
```
